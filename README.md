# Employee Management System

This is a full-stack Employee Management System.  
It allows you to **create**, **read**, **update**, and **delete** employees, along with search and pagination functionality.

---

## ðŸ§° Technology Stack

- **Frontend**: React.js with functional components and Hooks (`useState`, `useEffect`)  
- **Backend API**: Node.js + Express (endpoint paths under `/api/employees`)  
- **Database**: MongoDB (via Mongoose)  
- **Styling**: Bootstrap CSS for layout and UI components  
- **Routing**: React Router (for navigating to employee details)  
- **Notifications**: React-Toastify for success/error messages  
- **File Upload**: Support for profile images using `FormData` in the frontend and appropriate handling in the backend  
- **Pagination & Search**: Query parameters `search`, `page`, `limit` used for fetching employee lists
